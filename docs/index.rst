====================================
Welcome to MiniBlog's documentation!
====================================

MiniBlog is a small piece of software designed to make blogging easy and quick
without much of a hassle. The same principle applies to the software itself:
It tries to use as much as possible from other libraries to reduce own code.

MiniBlog uses Markdown for blog content and Persona for admin login to make
integration easy and writing articles uncomplicated.

To get started using MiniBlog head to the :ref:`Quickstart <quickstart>` section.
